Simplified SQS Wrapper and Async Worker manager.


Features:
- Simple interface. :white_check_mark:
- Promise based. :white_check_mark:
- ES6. :white_check_mark:
- Optimized async worker. :white_check_mark:

## Install

```sh
# Using npm
$ npm install wtsqs --save

# Or using yarn
$ yarn add wtsqs
```
